Publicação:
1) Suba a pasta inteira no GitHub.
2) No Netlify: Add new site > Import from GitHub > selecione o repo.
3) Build command: (vazio) | Publish directory: .

Checkout:
- Edite index.html e troque {{SEU_LINK_MERCADO_PAGO}} pelo seu link, OU
- acrescente ?checkout=SEU_LINK ao final da URL.
